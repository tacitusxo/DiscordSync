# DiscordSync

A desktop application built with Electron and React that synchronizes Discord server data and tasks.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

</div>

## About

DiscordSync is a desktop application designed to help manage and synchronize Discord server information with local task management. Built with modern technologies including Electron, React, TypeScript, and Webpack.

## Prerequisites

- Node.js >= 14.x
- npm >= 7.x

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/takikunxo/DiscordSync.git
cd DiscordSync
npm install
```

## Development

Start the application in development mode:

```bash
npm start
```

To run linting checks:

```bash
npm run lint
```

To run tests:

```bash
npm test
```

## Building

Build the application for your platform:

```bash
npm run package
```

This creates an executable in the `release/build` directory.

## Technologies

- **Electron** - Cross-platform desktop application framework
- **React** - UI library
- **TypeScript** - Type-safe JavaScript
- **Webpack** - Module bundler
- **Electron Builder** - Packaging and building
